INSTALLATION DU SCRIPT

-Installer GreaseMonkey (Extension du navigateur firefox)
- Drag / Drop le fichier VisualStruct.user dans firefox
- Accepter la demande d'installation
- Le script est installé, charger une page et vérifier que GreaseMonkey ainsi que le script sont bien activés

LANCEMENT DU SCRIPT

Une fois le chargement de la page web effectué, une bulle sera présente (normalement près du bord droit) sur la page web.
Elle permet via un double clic l'affichage de l'interface d'adaptation et, le lancement du processus d'extraction et d'annotation sur la page web courante
