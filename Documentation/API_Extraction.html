(static) DOMNodeDivision(DOMNode, nLevel, pool)

Fonction récursive réalisant : 1- Filtrage des noeuds non intéressants 2- Le repérage et l'annotation des Noeuds Atomiques Virtuels(Virtual Content Node) 3- la non-recopie d'un noeud "divisible", c'est a dire un noeud pouvant être omis dans l'arbre des structures visuelles, ses enfants se voyant adoptés par son parent. Elle parcourt les éléments de l'arbre DOM tant qu'elle ne rencontre pas de bloc visuel, si c'est le cas, le noeud contenant le bloc visuel est ajouté dans la variable pool passée en parametre.
Parameters:
Name Type Description
DOMNode Node noeud DOM à traiter
nLevel int Entier représentant la profondeur courante dans l'arbre des structures visuelles
pool List.<Node> Ensemble des noeuds du DOM qui seront frères dans l'arbre des Structures visuelles
